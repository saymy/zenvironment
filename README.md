Welcome to zenvironment!
===================


这是一个检测客户端设备的JS模块。

----------

如何使用zenvironment.js
-------------

> **node环境下使用**

 1. 安装:
 npm install --save zenvironment

 2. 引用:
 var zenvironment = require('zenvironment')
 
 or
 
 import 'zenvironment'

> **Demo **

   浏览器环境使用，下请查看文件中的Demo.html
 

> **使用和 API 说明**

  执行之后包括两个Obj对像：base（客户端最原始的设备信息，你可以自己进行加工）和info（客户端设备信息已进行一定的加工，可以直接应用到你的项目中）。
